# crm-backend-project

Project for Udacity's Golang course

This project implements a RESTful CRUD API server for a CRM (Customer Relationship Management) system.

# Usage

## Running the API server
To run the server, run:
```
go run 
```

## API Endpoints

* GET /customer/{id}
* GET /customers
* POST /customers
* UPDATE /customers/{id}
* DELETE /customers/{id}
